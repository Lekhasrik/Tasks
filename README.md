Day 1 — HTML Basics Page
Goal: Build a simple “About Me” webpage
Include:
- <!DOCTYPE html>, <html>, <head>, <title>, <meta charset="UTF-8">
- Headings (h1–h3), paragraphs
- One image with alt text
- <hr> separator

Day 2 — Lists + Links
Goal: Create a “My Favorites” page
Include:
- Ordered list (Top 10 items)
- Unordered list (Hobbies/skills)
- Definition list
- 5 links (external + internal anchors)

Day 3 — Tables
Goal: Create a Student Marksheet
Include:
- Table with Subject, Max Marks, Marks, Grade
- Use <thead>, <tbody>, <tfoot>
- Use rowspan or colspan
- Table caption

Day 4 — Forms Basics
Goal: Build a Registration Form
Include:
- text, email, password, number, date inputs
- radio buttons and checkboxes
- labels for inputs
- dropdown select
- submit and reset buttons

Day 5 — Advanced Forms
Goal: Job Application Form
Include:
- fieldset and legend
- file upload
- textarea
- range input
- required fields

Day 6 — Semantic Layout
Goal: Simple Blog Page
Include:
<header>, <nav>, <section>, <article>, <aside>, <footer>
Blog title, author, date
Navigation links
Recent posts list

Day 7 — Media Page
Goal: Media Gallery
Include:
- 6 images
- YouTube iframe
- audio element
- video element

Day 8 — Mini Website
Goal: 3-page website
Pages:
home.html
about.html
contact.html
Include navigation between pages

Day 9 — HTML Components
Include:
FAQ using <details> and <summary>
<progress> element
<meter> element
<blockquote>
<mark>

Day 10 — Final HTML Project
Create a College Event Website with:
Home section
Schedule table
Speakers section
Registration form
Contact section with Google map iframe
Footer with social links
